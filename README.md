# cool-maze
[Cool Maze - Push to Computer](https://coolmaze.io/) is a quick mobile-to-desktop share app, through matrix barcode.

![coolmaze illustration](https://user-images.githubusercontent.com/13508141/30317173-a2ab83b8-97a9-11e7-9e55-3fea754d0acd.png)

This repo contains (open source):
- Web client for [Cool Maze - Push to Computer](https://coolmaze.io/)

This repo does not contain (closed source):
- Cool Maze - Push to Computer Backend services
- [Cool Maze - Push to Computer Android app](https://play.google.com/store/apps/details?id=com.bartalog.coolmaze)
- [Cool Maze - Push to Computer iOS app](https://itunes.apple.com/us/app/cool-maze/id1284597516?mt=8)

This repo does not contain (open source):
- The backend and frontend of the reverse use case [Hot Maze](https://hotmaze.io/), which have moved to [this repo](https://github.com/Deleplace/hot-maze).

The [Issue tracker](https://github.com/Bartalog/cool-maze/issues) welcomes bugs of all components of Cool Maze - Push to Computer.
